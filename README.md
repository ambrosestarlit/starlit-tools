# 🎬 Scene*Writer

シーン構成型シナリオエディター - ボイスドラマや音声作品のシナリオ執筆に特化したWebアプリケーション

## 概要

Scene*Writerは、音声作品制作者向けに開発されたブラウザベースのシナリオエディターです。最大300シーンまでの管理が可能で、キャラクター管理、時間計算、効果音挿入など、音声作品制作に必要な機能を網羅しています。

## 主な機能

### 📝 基本機能
- **プロジェクト管理**: 新規作成、保存、読込、エクスポート機能
- **自動保存**: ブラウザのlocalStorageを使用した自動保存
- **複数形式対応**: JSON（.vwp）形式とTXT形式での入出力

### 🎭 キャラクター管理
- キャラクター登録・管理システム
- ワンクリックでキャラクター名を挿入
- 台詞形式の切り替え（「」形式 / ：形式）
- 特殊文字ボタン（…、っ、ッ、♡、〜、ー など）
- カット表記ボタン（【カットA】〜【カットD】）

### 🎬 シーン管理
- 1〜300シーンまでの管理
- シーン別の目標分数設定
- 自動文字数カウントと推定時間計算（240文字=1分）
- シーン別あらすじ機能
- シーンリストによる簡単な切り替え

### 🔊 音声効果・演出
- ト書き、現在地の挿入
- 効果音（SE）、BGM、環境音の指定
- R18コンテンツ用機能（喘ぎ声、アダルト効果音）
- 創作スロット（ランダムアイデア生成）

### 📊 テキスト処理
- 文字置換機能
- 台詞への自動連番付与
- リアルタイム文字数カウント
- 推定尺（再生時間）の自動計算

### 📅 プロジェクト情報管理
- 作品タイトル設定
- 執筆期日の設定（カレンダー機能）
- 全体あらすじ記入
- 自由メモ機能

## 使い方

1. **プロジェクトの開始**
   - 「新規」ボタンでプロジェクトを作成
   - 作品タイトルと執筆期日を設定

2. **キャラクターの登録**
   - 「キャラ管理」からキャラクターを追加
   - エディター上部にキャラボタンが自動生成

3. **シナリオの執筆**
   - シーンを選択して執筆開始
   - キャラボタンや特殊文字ボタンで効率的に入力

4. **データの管理**
   - 「上書き保存」で定期的にバックアップ
   - 「TXT書出」で完成原稿を出力

## 時間計算について

文字数から自動的に推定尺（分数）を計算します：
- 0〜239文字 → 0分
- 240〜479文字 → 1分
- 480〜719文字 → 2分
- 720〜959文字 → 3分
- 以降、240文字ごとに1分ずつ加算

## 動作環境

- モダンブラウザ（Chrome、Firefox、Edge、Safari）
- JavaScript有効
- localStorageが使用可能な環境

## ノートパソコンでの使用

ノートパソコンで使用する場合は、ブラウザの表示倍率を調整すると使いやすくなります：
- **推奨倍率**: 125%〜150%
- **拡大**: Ctrl/Command + プラス（+）キー
- **縮小**: Ctrl/Command + マイナス（-）キー
- **リセット**: Ctrl/Command + 0（ゼロ）キー

## インターフェース

- **チョコレート配色**: 目に優しいブラウン系の配色
- **レスポンシブデザイン**: ウィンドウサイズに応じて自動調整
- **3カラムレイアウト**: シーンリスト、エディター、サイドパネル

## データ形式

- **プロジェクトファイル**: .vwp形式（JSON）
- **エクスポート**: .txt形式でのテキスト出力
- **インポート**: .vwpまたは.json形式に対応

## トラブルシューティング

- **プロジェクトが開けない場合**: ファイル形式が.vwpまたは.jsonか確認
- **文字数が更新されない場合**: ブラウザをリロード
- **保存したファイルが見つからない場合**: 「読込」メニューから確認
- **旧バージョンのデータ**: Cut-表記は自動的にScene-表記に変換
